Qdrant Multitenancy

Single collection, is_tenant: true on tenant_id. Per-tenant HNSW sub-graphs (m=0, payload_m=16). All queries scoped to tenant.

Qdrant Edge

Embedded vector search via Python bridge. No external Qdrant server. Runs in-process alongside the AI agent.

NVIDIA Embeddings

nv-embedqa-e5-v5 (1024 dims). Separate input_type for queries vs passages.

Access Control

Payload-based ACL: access_list field per point. Denied access triggers alerts stored in a separate collection.
