NemoClaw Family Hub

Shared AI memory with role-based access control, powered by Qdrant + NVIDIA

👨

Dad

ADMIN / OWNER

Full access. Can store memories, grant/revoke access to others, and receive alerts when someone is denied.

Agent: dad
👧

Daughter

FAMILY

Can access family-shared memories. Can store her own. Cannot see dad's private memories unless granted.

Agent: daughter
👩‍🍼

Babysitter

GUEST

Limited access. Cannot see family memories by default. Must be explicitly granted access. Owner gets alerted on denied requests.

Agent: babysitter

Demo Flow

  1. Dad: "Remember the wifi password is NemoClaw2026!" (stores as family-visible)
  2. Daughter: "What's the wifi password?" (gets it, she's family)
  3. Babysitter: "What's the wifi password?" (DENIED, dad gets an alert)
  4. Dad: "Check my alerts" (sees babysitter's request)
  5. Dad: "Give the babysitter access to the wifi password"
  6. Babysitter: "Check my alerts" (sees she now has access)
  7. Babysitter: "What's the wifi password?" (gets it this time)
